## 1. השלמת התוכן בדף הבית (המשך תרגום)
- [ ] רשת הדוגמאות (EXAMPLES) - כותרות ותגיות ("בוטיק אופנה", "מסחר" וכו')
- [ ] רשימת המקצועות (PROFESSIONS) - "חנויות ובוטיקים", "יופי וטיפוח" וכו'
- [ ] שלבי "איך זה עובד" (HOW_STEPS) - כותרות A/B/C והתיאורים
- [ ] כיתובי תמונות התהליך (PROCESS_IMGS)
- [ ] סקשן "כל עסק. אתר אחד. 5 דקות."
- [ ] סקשן "כל הכלים לכל תחום" - טקסט המשנה והפריטים
- [ ] `Footer` - לוודא שכל הטקסטים עוברים דרך t()

## 2. עמוד המחירים (PricingSection)
- [ ] **לקבל ממוטי את מודל המחירים האמיתי** - מה כלול ב-69 ₪, ומה הפיצ'רים הנפרדים (CRM וכו') ומחיריהם
- [ ] לבנות מחדש את PricingSection למחיר יחיד אמיתי (69/$28/€28) במקום 3 המדרגות המזויפות
- [ ] לתרגם אותו ל-5 השפות

## 3. שאר עמודי השיווק (ניתוב שפה + תרגום)
- [ ] `/templates` `/contact` `/help` `/register` `/login` `/terms` `/privacy` `/accessibility`
- [ ] ניתוב `/en/templates` וכו' (נתיבים מקוננים) - כרגע רק דף הבית ממופה
- [ ] מתג השפה בעמודים פנימיים שיעדכן URL בלי 404 (תלוי בנתיבים המקוננים)
- [ ] רינדור-זחלן ל-edge גם לעמודים האלה (כמו שיש לדף הבית)

## 4. איכות ו-RTL/LTR
- [ ] **הגהת דובר-שפת-אם** לערבית/צרפתית/רוסית לפני השקה ציבורית (הטיוטות הן AI)
- [ ] audit RTL מול LTR - החלפת `mr-/ml-/text-right` קשיחים ב-properties לוגיים כדי שאנגלית/צרפתית/רוסית ייראו נכון (הן LTR באתר שבנוי RTL-first)
- [ ] `/` לשמור כעברית קנונית + הפניה גאוגרפית ל-`/en` וכו' (עידון; כרגע `/` מסתגל לפי localStorage)

## 5. ניקיון SEO (מהדוח המקורי)
- [ ] ליישר את meta של דף הבית בין 3 המקורות (index.html סטטי / edge / SEOHead) - כרגע ה-edge מרחיב מילות מפתח אבל הסטטי עדיין צר ("אתר מכירתי")
- [ ] להסיר `/login` מהסיטמאפ (עמוד דק)
- [ ] להוסיף חלופות שפה (hreflang) גם בתוך הסיטמאפ
- [ ] לאכוף CSP (מ-Report-Only ל-Content-Security-Policy)
- [ ] og-image לכל שפה (אופציונלי) / לוודא 1200x630
