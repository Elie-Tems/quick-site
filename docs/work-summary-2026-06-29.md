# ⏳ מה מחכה לך (פעולות שלך)
1. **CRM** - קרא את [docs/crm-plan.md](crm-plan.md) ותגיד הערות + מי הלקוח העיקרי (קמעונאות מול B2B) כדי למקד את הבנייה.
2. **GitHub secret** - הוסף `SUPABASE_ACCESS_TOKEN` ב-GitHub → פריסת פונקציות תהיה אוטומטית. אחר כך **לבטל את ה-PAT הזמני**.
3. **כשiCount יתחבר** - לכבות `ALLOW_UNVERIFIED_PUBLISH` + `VITE_PUBLISH_SKIP_PAYMENT` (החזרת תשלום), ולספק קישורי תשלום (₪149 תגי מעקב, ₪14 ביקורות).
4. **Google Maps API key** (`GOOGLE_MAPS_API_KEY`) - לביקורות Google.
5. **לבדוק מהירות התמונות** - אם gpt-image-1 איטי מדי (~10-20ש'), אפשר quality "low" או Google ישיר (~5ש').
6. **מיקרופון** - לאשר הרשאה פר-אתר ב-Chrome (🔒 ← מיקרופון ← אפשר).
7. **לרענן** את ICOUNT_WEBHOOK_SECRET (צד שלך) + לבטל PAT ישן.

## הערות (לידיעה, לא דחוף)
- **expire-subscriptions** ניתן להפעלה פומבית (Low) - לא שיניתי כדי לא לשבור את ה-cron באמצע הלילה; פתרון: להגדיר CRON_SECRET ולעדכן את ה-cron יחד.
- מומלץ לוודא שמדיניות ה-RLS לאדמין מגבה את שמירת ה-MFA בצד-לקוח.
