# Changelog

## גרסה נוכחית — שלב המוצרים (2026-06-28)

### שלב הוספת מוצרים — עיצוב מחדש מלא

הפחתנו את StepProducts מ-1,300 שורות של טפסים כבדים ל-UI נקי עם 3 שיטות כניסה:

#### הוספה מהירה
- שדה שם + מחיר בשורה אחת, תיאור בשורה שנייה
- לחיצה על Enter מוסיפה מוצר מיד
- אין טפסים, אין מודאלים — המוצרים נוספים ישירות לרשימה

#### ייבוא קובץ (3 אפשרויות)
- **אקסל / CSV** — גרירה או בחירת קובץ, תצוגה מקדימה לפני ייבוא, כפתור הורדת תבנית מוכנה
- **PDF קטלוג** — העלאת PDF, חילוץ טקסט client-side עם pdfjs, פרסור עם AI
- **קישור לאתר** — הדביקו URL, הפונקציה סורקת ומחלצת מוצרים אוטומטית

#### הכתבה קולית
- הקלטה עם MediaRecorder API
- תמלול עם Whisper ופרסור עם GPT-4o-mini
- תצוגה מקדימה לפני ייבוא

### Supabase Functions חדשות
- **`parse-catalog-products`** — מקבל טקסט מ-PDF או URL, מחזיר רשימת מוצרים מובנית
- **`transcribe-products`** — מקבל אודיו base64, מתמלל עם Whisper ומפרסר עם GPT

### תיקוני ניווט ב-Onboarding
- תוקן באג שגרם לכפתורי "הבא" להיעלם לחלוטין ב-StepIdentity, StepContact, StepVisuals (`showPreview={false}` + `showSave={false}`)
- שלב המוצרים עבר ל-bar ניווט **דביק** (sticky) — תמיד גלוי בתחתית המסך ללא גלילה
- כפתור "המשך" / "דלג בינתיים" כ-primary action, "חזרה" כ-secondary

### שיפורים קטנים
- הוסרה התווית "(אופציונלי)" מ-placeholder התיאור
- תיקון worker URL של pdfjs-dist — מ-CDN חיצוני לייבוא Vite-native (יציב יותר)
- הוספת `codepage: 65001` ל-XLSX.read לתמיכה נכונה ב-CSV בעברית
- כפתור "ייבוא קטלוג" שונה ל-"ייבוא קובץ"
